Create sub-domains from current website content.

== Description ==

Xenial: Sub-domain SEO is designed to get all the content from your website and create a custom one page website.

Functions:
- Logs into cpanel and creates subdomain on server.
- Pulls all the content from the website



== Installation ==

Xenial SDS is easy to install and use.

1. Upload Xenial SDS to the /wp-content/plugins/ directory
2. Unzip the file
3. Activate Xenial SDS
